# Juvo API

Backend API de Juvo para gestion de patentes, tipos de vehiculo y reportes.

## Stack

- Node.js
- TypeScript
- Hono
- MySQL (mysql2)

## Instalacion

```bash
npm install
```

## Desarrollo local

```bash
npm run dev
```

API en:

```text
http://localhost:3000
```

## Build y ejecucion

```bash
npm run build
npm start
```

## Variables de entorno

El proyecto usa variables de DB desde entorno (ejemplo en .env.local):

- DB_HOST
- DB_PORT
- DB_NAME
- DB_USER
- DB_PASS
